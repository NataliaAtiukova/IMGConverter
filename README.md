# IMGConverter

iOS project built with Swift/Xcode.

## Status

Portfolio/WIP project.

## Tech

- Swift
- Xcode project structure

## Run locally

1. Clone the repository:
```bash
git clone https://github.com/NataliaAtiukova/IMGConverter.git
cd IMGConverter
```
2. Open `IMGConverter.xcodeproj` in Xcode.
3. Build and run on simulator/device.
